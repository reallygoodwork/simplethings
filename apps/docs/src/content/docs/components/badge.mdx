# Badge


A badge component

## Props

| Name | Type | Default | Options |
| ---- | ---- | ------- | ------- |
| label | string |  |  |
| variant | string | primary | grey, primary, alt, accent, success, warning, error, default |
| rounded | boolean |  |  |
| vibe | string | none | outlined, soft, none |
